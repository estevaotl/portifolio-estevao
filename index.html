MUITO PRAZER, SOU ESTÊVÃO TIENGO LEITE.

Minha trajetória profissional inclui tanto o desenvolvimento de sistemas robustos quanto a liderança de equipes técnicas, sempre com uma forte capacidade analítica, boa comunicação e habilidades para resolver problemas complexos.

MEU PORTFÓLIO.

Match Serviços

Plataforma web que permite aos usuários se registrarem e buscarem prestadores de serviços com base na proximidade geográfica. O sistema calcula a distância entre o usuário e os profissionais disponíveis e oferece um portfólio visual com imagens dos trabalhos realizados. Após selecionar um prestador, o usuário pode contatá-lo via WhatsApp para iniciar a contratação.

Projeto Docker

Este projeto, desenvolvido como um teste técnico para a EzOps, utiliza NodeJS para enviar mensagens a um banco de dados não relacional.
Inclui automação no GitHub para atualizar o container automaticamente, garantindo que as modificações fiquem disponíveis online sem necessidade de pull no servidor. A aplicação foi criada com Docker e hospedada na AWS.

Criptografia RSA

Aplicação para criptografar e descriptografar textos utilizando criptografia e métodos RSA.
Recebe arquivos criptografados e descriptografados e faz o processo inverso.

Captura de dados do cnpj dos sites

Projeto realizado para captura de informações juridicas dos sites, como CNPJ, endereço etc.
Após a captura desses dados, validamos eles na receita federal, para verificar se as informações batem com o que foi cadastrado, a fim de evitar que os consumidores caiam em golpes.
Por fim, é gerado um html de saída com um score de acordo com a quantidade de dados válidos.
